## 🎯 Objetivos da Disciplina
O foco deste curso é capacitar o aluno a garantir a qualidade de software através de estratégias de verificação e validação automatizadas. Ao final do semestre, você será capaz de:

* Criar **Testes de Unidade** para validar regras de negócio isoladas.
* Utilizar **Dublês de Teste (Mocks)** para isolar dependências.
* Implementar **Testes de Integração** em aplicações Spring Boot.
* Desenvolver **Testes de Aceitação (E2E)** automatizando interfaces web.
* Analisar **Métricas de Cobertura** e qualidade de código.

---

## 🛠️ Stack Tecnológica

As aulas e projetos utilizarão as ferramentas padrão de mercado para o ecossistema Java:

* **Linguagem:** Java 
* **Framework:** Spring Boot 
* **Test Runner:** JUnit 5 (Jupiter)
* **Asserções:** AssertJ
* **Mocks:** Mockito
* **Testes de API:** MockMvc
* **Automação Web:** Selenium WebDriver
* **Banco de Dados (Testes):** H2 Database

---

## 📚 Conteúdo Programático

1.  **Fundamentos:** Cultura de qualidade, Pirâmide de Testes e TDD.
2.  **Testes de Unidade:** JUnit 5, Ciclo de vida (AAA), Asserções.
3.  **Isolamento:** Uso de Mockito (`@Mock`, `@InjectMocks`, `when/thenReturn`).
4.  **Testes no Spring Boot:** `@SpringBootTest`, `@DataJpaTest`, `@MockBean`.
5.  **Testes de Integração:** Validando a comunicação entre camadas e APIs.
6.  **Testes E2E:** Automação de UI com Selenium.
7.  **Métricas:** Análise de cobertura com JaCoCo.
